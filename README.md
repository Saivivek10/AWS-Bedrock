# AWS-Bedrock

## Overview
This project demonstrates how to use AWS Bedrock to interact with various language models like Claude, Llama2, and Titan Embeddings. It enables users to chat with PDF documents by generating embeddings, storing them in a vector database, and retrieving relevant information using language models.

## Features
- Generate embeddings using Amazon Titan Embeddings.
- Store and retrieve vector embeddings using FAISS.
- Interact with language models like Claude and Llama2.
- Process and query PDF documents.

## Prerequisites
- AWS account with access to Bedrock services.
- Python 3.8 or higher.
- Install the required dependencies listed in `requirements.txt`.
